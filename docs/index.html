pitv — Waiting Room TV Kiosk

A Raspberry Pi application that turns a waiting room TV into an interactive kiosk. Guests scan a QR code with their phone, enter a PIN, and pick a video to play on the shared display — no app required, no internet needed.

How It Works

Step 1

Scan the QR Code

The TV shows a QR code and 4-digit PIN while idle. Any phone camera works.

Step 2

Enter the PIN

First to enter the PIN becomes the controller for this round.

Step 3

Pick a Video

Browse the grid and choose. It plays on the big screen immediately.

Step 4

Watch Together

Video plays to completion. A fresh PIN is issued for the next round.

Features

  • ✓ No app required — everything runs in the phone's browser
  • ✓ Choose wisely — videos play to the end, no one can stop someone else's pick
  • ✓ Free Play mode — skip the PIN, anyone can pick or stop; great for personal use
  • ✓ Admin panel — force-reset, override playback, reboot, toggle modes
  • ✓ Easy content management — drop a video and poster image into a named folder under media/
  • ✓ Runs locally — no cloud, no subscriptions, no internet required

Installation

Flash a Pi with Raspberry Pi OS Bookworm (set username to pitv), then:

$ git clone https://github.com/alwaysunpredictable/pitv.git waiting-room
$ cd waiting-room
$ printf 'APP_SECRET=\nADMIN_PASS=changeme\n' > config.env # change ADMIN_PASS; APP_SECRET is auto-generated
$ sudo bash setup.sh && sudo reboot

The setup script handles packages, virtual environment, systemd services, and autologin. See the full README for details.
